# ОКСНАУ-Кіоск

Веб-додаток кіоску для коледжу ОКСНАУ. Додаток створений на Vue 3 + Vite та призначений для інтерактивних терміналів (кіосків) в приміщенні коледжу.

## Функціональність

- **Карта коледжу** - інтерактивна схема поверхів будівлі
- **Розклад занять** - перегляд розкладу занять
- **Спеціальності** - інформація про навчальні спеціальності
- **Абітурієнти** - інформація для вступників
- **Укриття** - схема та інформація про укриття
- **Контакти** - контактна інформація коледжу

## Технології

- Vue 3 (Composition API)
- Vite
- TypeScript
- Vue Router
- Pinia

## Налаштування проекту

```sh
npm install
```

### Компіляція та гаряче перезавантаження для розробки

```sh
npm run dev
```

### Перевірка типів, компіляція та мініфікація для продакшену

```sh
npm run build
```
